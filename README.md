# K&K Consultora

Sistema centralizado de gestión de conocimiento, proyectos y soluciones de K&K Consultora.

## Objetivos Empresariales

- **Gestión de Proyectos**: Entregar proyectos con metodología estructurada, trazabilidad y mejora continua
- **Mejora de Procesos**: Diagnosticar, rediseñar y optimizar procesos operativos de clientes

## Estructura del Repositorio

```
K-K/
├── empresa/              # Contexto estratégico de la consultora
├── proyectos/            # Proyectos activos, pausados y cerrados
├── investigaciones/      # Research, frameworks y metodologías
├── base-conocimiento/    # Patrones, lecciones y herramientas reutilizables
├── plantillas/           # Templates estándar para entregables
└── metricas/             # KPIs de la consultora y proyectos
```

## Cómo Usar Este Sistema

1. **Nuevo proyecto** → copiar `plantillas/proyecto/` → crear en `proyectos/activos/`
2. **Nuevo hallazgo/patrón** → documentar en `base-conocimiento/patrones/`
3. **Nueva investigación** → agregar en `investigaciones/`
4. **Al cerrar proyecto** → mover a `proyectos/cerrados/` y extraer lecciones
